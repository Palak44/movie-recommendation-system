🎬 Movie Recommendation System

📌 Project Overview
Built a content-based movie recommendation system that suggests similar movies using cosine similarity on movie metadata.

🛠 Tech Stack
Python
Pandas
Scikit-learn
NLP (CountVectorizer)
Cosine Similarity

⚙️ How It Works
Data preprocessing
Feature engineering
Text vectorization
Similarity matrix generation
Recommendation function

🚀 Example
Input: Avatar
Output: 5 similar recommended movies

📂 Dataset
TMDB 5000 Movies Dataset (Kaggle)
